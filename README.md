# Financial_crisis_and_resilience_in_Euro_area

We use a structural gravity model to understand how the 2008 financial crisis affected international trade, with a specific focus on euro area resilience.

## Datasets

Gravity_M2IT.dta (from https://faridtoubal.com/teaching.html)
If necessary, Gravity Portal datasets (https://www.usitc.gov/data/gravity/gravity_portal)
